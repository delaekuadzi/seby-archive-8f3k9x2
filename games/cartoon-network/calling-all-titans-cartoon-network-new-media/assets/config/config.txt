
To Configure PowerPlay games, you must include the screen resolution & the game Size. See below...

To include a feature, prepend an asterisk next to it (*). Remove the asterisk or put a minus sign (-) next to it to indicate that this game should not use a given feature. Or you can just leave the feature out... but ScreenSize & GameSize must be included.

The Gamesplash Screen time can either be specified as number of milliseconds or set to auto for .swf driven control (see documentation).

To Specify the size of the Instructions.swf, set 'InstructSize=" to the desired widthXheight. If 'InstructSize' is not set, the Instructions.swf will be displayed FullScreen.



-- Menu and Game Options

*MonitorSize= 640x480
*GameSize= 600x400
*Instructions
*InstructSize= 302x252
*SplashTime= auto
-SaveGames
-HighScores
-Billboard


-- Scoring Options

-PostScoresStayInGame
-ReverseScores
-NamesProvided
-HighScoreDelay= 180
